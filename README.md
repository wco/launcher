# WOW-COLOMBIA Launcher

Launcher oficial de [WOW-COLOMBIA](https://www.wow-colombia.com) — requerido para jugar.

## Descarga

| Archivo | Descripción |
|---------|-------------|
| [wow-colombia.zip](https://github.com/wco/launcher/releases/download/v3.4.1/wow-colombia.zip) | **Recomendado**: launcher comprimido (clave: `wowco`) |
| [wow-colombia.exe](https://github.com/wco/launcher/releases/download/v3.4.1/wow-colombia.exe) | Launcher directo (sin comprimir) |

> Contraseña del ZIP: `wowco`

## Instalación

1. Descarga `wow-colombia.zip` y ábrelo (contraseña: `wowco`).
2. Extrae `wow-colombia.exe` en la carpeta donde está tu `Wow.exe`.
3. Ejecuta `wow-colombia.exe`. Si Windows muestra **"Windows protegió tu PC"**, haz clic en **"Más información"** → **"Ejecutar de todos modos"**.
4. El launcher inicia el juego automáticamente y se actualiza solo en siguientes ejecuciones.

> Si tu antivirus bloquea el launcher, consulta la guía en [Errores Comunes](https://www.wow-colombia.com/post/errores-comunes) para agregar la exclusión manual.

## Auto-Actualización

Integrada desde v3.0.8. Cuando hay una nueva versión disponible, el launcher la descarga y aplica automáticamente mostrando un diálogo con notas de versión y barra de progreso.

## Advertencia de SmartScreen

Windows puede mostrar una alerta de SmartScreen al ejecutar el launcher por primera vez. Es normal para aplicaciones nuevas sin firma digital. Haz clic en "Más información" → "Ejecutar de todas formas".
